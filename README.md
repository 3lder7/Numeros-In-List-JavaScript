# 📊 Análise de Números
Este é um aplicativo simples em JavaScript que permite ao usuário adicionar números de 1 a 100, visualizar os números adicionados e obter informações estatísticas sobre eles.

## 🎯 Principais Funcionalidades
- **Adição de Números:** Permite adicionar números de 1 a 100 a uma lista.
- **Validação de Entrada:** Verifica se o número está dentro do intervalo permitido e se já foi adicionado.
- **Estatísticas:** Após finalizar, o aplicativo fornece:
  
  - Total de números adicionados.
  - O maior e o menor número informado.
  - A soma de todos os números.
  - A média dos números.
 
 ## 🎨 Design
- **Interface Limpa:** O design é simples e intuitivo, facilitando a interação do usuário.
- **Responsividade:** O layout se adapta a diferentes tamanhos de tela, proporcionando uma boa experiência em dispositivos móveis e desktops.

## 💾 Tecnologias Utilizadas
- ✅**HTML** Estrutura da página.
- ✅**CSS:** Estilização e design.
- ✅**JavaScript:** Lógica do aplicativo e manipulação do DOM.

## 📦 Como Usar
1. Clone o repositório:
```
git clone https://github.com/s3lder7/Numeros-In-List-JavaScript.git
```
2. Abra o arquivo modelo.html em um navegador.
3. Adicione números e clique em "Adicionar".
4. Finalize para ver as estatísticas.
